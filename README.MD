Бекенд для маркетплейса (интернет-магазина)
User_service:
- Создание пользователя
- Получение пользователя по id, email
- Получение всех пользователей 
- Удаление пользователя
- Смена имени, фамилии, email пользователя
- аутентификация и авторизация (в разработке)

Product_service:
- Добавление товаров, категории
- Удаление товаров, категории
- Измененение доступного количества товаров
- Получение товара, категории по id или всех товаров и категорий
- Изменение описания товара или категории

Cart_service:
- Добавление товара в корзину (хранятся Redis)
- Удаление товара из корзины
- Очистка корзины
- Получение текущей корзины

Order_service - в разработке
Recomendation_service

--- для каждого сервиса представлен пример файла .env, для запуска необходимо изменить URL-адреса

Запуск 

project > docker compose up --build

Сервисы работают по адресам:

User_service - localhost:8000

Product_service - localhost:8001

Cart_service - localhost:8002

Остановка 

project > docker compose down
